# Front-end | Componentização de campo de seleção

## O novo componente poderá ser reutilizado no processo de novos módulos e refatoração de antigos.Teremos ganhos na reutilização de trechos de código, no isolamento de contexto, na legibilidade do código, e na padronização do projeto.


:bulb: **Sugestão de consulta:** Arquivo `VSelect.vue`

### :pushpin: Descrição
O campo será utilizados nas telas do fluxo de tranferências, onde será realizada o cadastro e listagem de transferencias.

#### Itens para o componente
- As seguintes opções deverão ser passadas como props no componente pois serão mutáveis.
  - Receber um placeholder por parâmetro;
  - Recebe uma lista de oções para o drop do campo;
  - Deve emitir evento de change ao ser clicado;
  - Deve existir validaçao quando o campo esta com dados incorretos/vazio
